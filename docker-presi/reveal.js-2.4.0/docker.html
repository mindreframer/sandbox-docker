Docker installation


## install 3.8 kernel, if on Ubuntu 12.04
$ sudo apt-get update
$ sudo apt-get install linux-image-generic-lts-raring

# Add the PPA sources to your apt sources list.
$ sudo apt-get install python-software-properties
$ sudo add-apt-repository ppa:dotcloud/lxc-docker

# Update your sources
$ sudo apt-get update

# Install, you will see another warning that the package cannot be authenticated. Confirm install.
$ sudo apt-get install lxc-docker

Let's add Vagrant-Cachier

it will prevent us from redownloading packages/gems/rvm tarballs all the time


$ vagrant plugin install vagrant-cachier

for apt-cacher bucket use my branch

until merged into master branch 

cd ~/.vagrant.d/gems/gems/
rm -rf vagrant-cachier-0.2.0
git clone https://github.com/mindreframer/vagrant-cachier.git vagrant-cachier-0.2.0
cd vagrant-cachier-0.2.0 && git checkout origin/apt-cacher
rm -rf vagrant-cachier-0.2.0/.git
